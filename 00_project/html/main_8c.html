This is the project Street-fighter by Simon Thür and Lokman Mheni. The main aim is a multiplayer 1v1 fighter game through wifi connection. As an 'afterthought' the game also features a very basic single player mode.

Here is a list of where requirements can be found

(Only examples, not all references)

  1. Arm9 Arm7 processors both used
    • Arm9
      • Arrow keys: input.c
      • Permanent storage: memory.c
    • Arm7
      • Wifi: wifi.c
  2. Timers/Interrupts
    • Game countdown: graphics.c (manage_timer())
    • Delayed attack: game_controller.c (local_attack())
  3. Both graphics engines, at least 2BGs on one of them
    • Timer and healthbars: graphics.c (show_timer() and set_healthbars())
  4. Keypad at least one key used
    • User input: input.c (get_input())
  5. Touchscreen used
    • Select Singleplayer/multiplayer: input.c (get_touch_input())
  6. Sound (music and sound-effect)
    • Music and sound-effects: sound.c (play_music() and play_sound_effect())
  7. Secondary storage
    • Storing and retrieving high-score: memory.c (store_stats() and get_stats)
  8. Wifi
    • Transfer data between NDS: wifi.c
  9. Sprites
    • Player characters displayed as sprites: graphics.c (sprite_pos_remote())
