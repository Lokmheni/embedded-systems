◆ local_attack()

bool local_attack ( bool  special)

Execute an attack. return attack characteristic as WifiMsg format. This action will block player movements for a small delay (uses timer2 and ISR).

Note
This functions automatically calls local_attack_handler(u8* dmg_x, u8* dmg_y, u8* dmg) at the correct time (either instantly or after an interrupt delay).
Parameters
specialIf attack is supposed to be special attack
Returns
true if the attack is valid and will be transmitted
false if the attack is not legal at this time.
